# 🏟️ Python Arena Game

## 📖 О проекте

**Python Arena Game** — это консольная игра, написанная на Python.

Игрок выходит на арену и сражается с тремя противниками. В каждом бою необходимо принимать решения, следить за количеством здоровья и победить всех врагов, чтобы пройти игру.

---

## ✨ Возможности

- ⚔️ Пошаговая система боя
- 👹 Три противника с разными характеристиками
- ❤️ Система здоровья (HP)
- 💥 Система нанесения урона
- ✅ Проверка корректности ввода действий
- 🏆 Победа или поражение в зависимости от исхода боя

---

## 🛠 Использованные технологии

- Python 3

---

## 🚀 Как запустить

1. Установите Python 3.
2. Склонируйте или скачайте этот репозиторий.
3. Запустите файл:

```bash
python arena_game.py
```
